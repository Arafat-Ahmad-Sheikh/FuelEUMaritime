# FuelEU Maritime — Minimal Implementation

This repository contains a minimal backend and frontend scaffolding for the FuelEU Maritime assignment.

Quick start (backend):

```bash
cd backend
npm install
npm run dev
```

The backend exposes:
- `GET /routes`
- `POST /routes/:id/baseline`
- `GET /routes/comparison`
